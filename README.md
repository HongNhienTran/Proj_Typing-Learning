Typing Test App

## 🚀 Features

- **Real-time Stats:** Track your Words Per Minute (WPM), Accuracy (%), and elapsed time as you type.
- **Monkeytype-style Cursor:** Smooth, absolute-positioned blinking caret that glides naturally across characters.
- **Smart Color Feedback:** - `Gray` for pending characters.
  - `Black/White` (Light/Dark mode) for correctly typed characters.
  - `Red with light red background` for typos, making mistakes instantly visible.
- **Time Limit Mode:** Automatically stops the test and freezes input when the timer hits the limit (e.g., 30 seconds).
- **Interactive Controls:** Easily **Pause**, **Resume**, or **Retry** the test at any time.
- **Detailed Results Page:** Provides a beautiful grid summary of your performance (WPM, Accuracy, Total Score, Duration, and Mistakes) with a "Back to Home" redirection.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## 📦 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HongNhienTran/Proj_Typing-Learning.git
   cd typing-academy

```

2. **Install dependencies:**
```bash
npm install
# or
yarn install
# or
pnpm install

```


3. **Run the development server:**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev

```
4. Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) with your browser to see the result.
